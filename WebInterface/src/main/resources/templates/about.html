WHAT DOES THIS INTERFACE ENTAIL

Quantifying and Classifying the digital (Social Media) footprint of South African telecommunications companies before and during the COVID-19 Pandemic
Cornerstone group project for COMP301 (Software Design) module

HOW WE HAVE FORMED OUR PROJECT

Our group has taken to Twitter to gather information about 4 Industrial Revolution companies, namely Rain, Telkom, Afrihost and MTN.
We used python scraping tools to store all tweets related to these companies before and during the pandemic. We then created a web interface using Java and its Spring Boot tools in order to display some of the statistics we found. This interface has user friendly graphs to show the data found.
