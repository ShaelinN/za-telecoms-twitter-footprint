TOPIC

Quantifying and Classifying the digital (Social Media) footprint of South African telecommunications companies before and during the COVID-19 Pandemic
This is the cornerstone group project for the COMP301 (Software Design) module


WHAT DOES THIS INTERFACE ENTAIL

Allows for the user to select queries that can be performed on the data, and displays the results (both tabular data and some charts)


HOW WE HAVE DEVELOPED OUR PROJECT

Data Acquisition - We set up a Python Scraper to extract tweets from Twitter using the Twint API from the 4 South African telecommunications companies (Telkom, Afrihost, MTN, and Rain - over the specified time period
Database - We selected Neo4j as our NoSql Graph Database, and learned the Cypher query language to create the database and perform queries on it
Web Interface - We created it using the IntelliJ IDEA IDE and we using the Spring Framework as our Web Framework along with Java for the Controllers, Cypher for the queries to the database, and JavaScript, HTML, and CSS for the Web Pages
